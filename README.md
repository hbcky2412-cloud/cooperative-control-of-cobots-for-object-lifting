# Cooperative Control of Cobots for Object Lifting

This project studies the coordinated control of multiple collaborative robots (cobots) for lifting and manipulating a heavy object. The objective is to design a control strategy that allows multiple robots to work together safely and efficiently.

## Overview

Collaborative robots are increasingly used in industrial environments where humans and robots share the workspace. When manipulating heavy objects, multiple robots must coordinate their motion and forces to ensure stable and safe manipulation.

This project investigates the modeling and control of a cooperative robotic manipulation system.

## Objectives

- Model the cooperative manipulation system
- Analyze the interaction between multiple robots and the object
- Design a coordinated control strategy
- Evaluate the performance of the cooperative manipulation task

## Methodology

The project includes the following steps:

1. Modeling the robotic system and manipulated object
2. Deriving the system dynamics
3. Designing a cooperative control strategy
4. Simulating the lifting and manipulation task
